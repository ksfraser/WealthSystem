## Design Overview

### Design Principles
1. **Information Density**: Show maximum relevant data without overwhelming
2. **Contextual Help**: Help always one click away
3. **Visual Hierarchy**: Most important info prominent
4. **Responsive**: Works on desktop, tablet, mobile
5. **Performance**: Real-time updates without lag

### Component Breakdown

#### Left Sidebar: Indicators & Patterns
- **Indicators Section**: Add/remove technical indicators
- **Patterns Section**: View detected candlestick patterns
- **Strategies Section**: Apply strategy overlays
- **Collapsible**: Can hide for more chart space

#### Main Chart Area
- **Primary Chart**: Candlestick, line, or OHLC
- **Timeframe Selector**: 1D, 5D, 1M, 3M, 1Y, 5Y, Max
- **Drawing Tools**: Trendlines, support/resistance
- **Annotations**: Pattern markers, signals

#### Indicator Panels (Below Chart)
- **Stacked Panels**: Each indicator in own panel
- **Resizable**: Drag to adjust height
- **Minimize/Close**: [?] for help, [Ã—] to remove
- **Settings**: Click gear icon to configure

#### Right Sidebar: Order Entry
- **Order Type**: Market, limit, stop, stop-limit
- **Price/Quantity**: Input fields
- **Risk Calculator**: Auto-calculate position size
- **Quick Actions**: Buy/sell buttons

## Interaction Patterns

### Tooltip Behavior
1. **Desktop**: Hover shows quick tooltip
2. **Desktop+Shift**: Hover+Shift shows detailed tooltip
3. **Desktop+Click**: Click (?) icon opens help modal
4. **Mobile**: Tap (?) icon opens help modal (no hover)

### Help Modal Transitions
- **Open**: Fade in background overlay (300ms), slide modal from right (250ms)
- **Close**: Slide out to right (200ms), fade out overlay (200ms)
- **Swipe**: On mobile, swipe right to close

### Chart Interactions
- **Pan**: Click+drag (desktop), touch+drag (mobile)
- **Zoom**: Scroll wheel (desktop), pinch (mobile)
- **Crosshair**: Hover shows values at cursor
- **Annotations**: Click pattern marker to see details
